# KIPP St. Louis - Data & Compliance Command Center

A comprehensive educational data management and compliance tracking system demonstrating full-stack web development capabilities.

## Live Demo Features

### Data Management & Analytics
- **Real-time Data Quality Monitoring** - Validation of 1,247+ student records with automated issue detection
- **SQL Query Builder** - In-browser SQLite database (SQL.js) with real query execution and Excel export
- **Strategic KPIs** - Achievement tracking, enrollment forecasts, and financial margin analysis

### Compliance & Reporting
- **DESE/MOSIS Integration** - Missouri state reporting with fixed-width format generation
- **Compliance Tracker** - Deadline management, task tracking, and automated notifications
- **Truancy Interventions** - Missouri RSMO 167.031 compliance with tiered letter generation

### Advanced Visualizations
- **3D Analytics Lab** - Interactive student risk visualization (Plotly.js)
- **Force-Directed Networks** - Intervention pathway mapping (D3.js)
- **Sankey Diagrams** - Cohort flow and student progression tracking
- **Geographic Mapping** - School performance visualization
